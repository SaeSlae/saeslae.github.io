HTML Reference

HTML TABLE

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table data within the row. Each seperated by new "td" within brackets
<tfoot> Table Footer The set of rows defining the foot of the table

Table Attributes

Attribute Name Description
<colspan> Column Span Defines how many columns an element should span.
<rowspan> Row Span Defines how many rows an elemnt should span.
